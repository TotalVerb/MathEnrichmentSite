Sum and Product Rules

Problems marked with an asterisk (*) are considered to be challenge problems. You should still try them, but we will discuss them in class.

Problem 1: Rolling Two Dice

I roll two different six sided dice (obtaining an ordered pair of two integers from 111 to 666).

  1. How many combinations are there which sum to at least 888?

  2. How many combinations are there where both numbers are even?

  3. How many combinations are there where the product is even?

Solution

Part 1

We can count the number of allowable conbinations for each possible roll of the first die:

  • If the first die is 111, there are no combinations that work.

  • If the first die is 222, there is one combination possible: the second die must be 666.

  • If the first die is 333, there are 222 combinations possible: the second die must be 555 or 666.

  • If the first die is 444, there are 333 combinations possible: the second die must be between 444 and 666.

  • If the first die is 555, there are 444 combinations possible: the second die must be between 333 and 666.

  • If the first die is 666, there are 555 combinations possible: the second die must be between 222 and 666.

In total, there are 1+2+3+4+5=151 + 2 + 3 + 4 + 5 = 151+2+3+4+5=15 possible combinations.

Part 2

There are 333 even numbers possible for the first die, and 333 even numbers possible for the second die. Therefore, there are 3×3=93 \times 3 = 93×3=9 total combinations where both numbers are even.

Part 3

Another way to approach this problem is by considering all 6×6=366 \times 6 = 366×6=36 possible combinations and deducting the ones where the product is odd. If the product is odd, that means both the first die and second die have to display an odd number. There are 3×3=93 \times 3 = 93×3=9 such combinations. Thus, there are 36−9=2736 - 9 = 2736−9=27 combinations where the product is even.

Problem 2: Teamwork*

There are 101010 people in a class. (For convenience, we can name them {1,2,…,10}\{1, 2, \dots, 10\}{1,2,…,10}.) How many different ways are there to make a team of 444 people? (For instance, {1,4,8,9}\{1, 4, 8, 9\}{1,4,8,9} is a possible team. The order we list the people does not matter; {4,9,1,8}\{4, 9, 1, 8\}{4,9,1,8} represents the same team).

Hint: First, calculate the number of ways there are to make an ordered team of 444 people, where the order is important. For each unordered team, how many ways are there to make ordered team with the same set of people? We can divide these two numbers to get the number of unordered teams possible.

Solution

There are 10×9×8×7=504010 \times 9 \times 8 \times 7 = 504010×9×8×7=5040 ways to make an ordered team. (This is because there are 101010 choices for the first teammate, but then only 999 choices for the second one since we cannot choose the same person twice. Then there are only 888 choices for the third one, and 777 choices for the fourth one.)

For each unordered team, there are 4×3×2×1=244 \times 3 \times 2 \times 1 = 244×3×2×1=24 ways to put an order on that team. (There are 444 choices for the first teammate, 333 choices for the second teammate, 222 choices for the third teammate, and 111 choice for the fourth teammate).

Therefore the number of ordered teams is 242424 times the number of unordered teams. Hence, the number of unordered teams is 5040÷24=2105040 \div 24 = 2105040÷24=210.
