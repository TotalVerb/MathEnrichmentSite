Problem 1: Counting Whole Numbers

All ranges in this document are inclusive (i.e. they include both of the endpoints). Make an exhaustive list to solve the problems below.

  1. How many whole numbers between 111 and 101010 are even?

  2. How many whole numbers between 111 and 101010 are odd?

  3. How many whole numbers between 111 and 101010 are divisible by 333?

  4. How many whole numbers between 111 and 101010 are divisible by 555?

  5. How many whole numbers between 111 and 101010 are not divisible by 777?

Solution

  1. There are 555: 222, 444, 666, 888 and 101010.

  2. There are 555: 111, 333, 555, 777 and 999.

  3. There are 333: 333, 666 and 999.

  4. There are 222: 555, 101010.

  5. There are 999: 111, 222, 333, 444, 555, 666, 888, 999 and 101010.

Problem 3: Counting Pairs*

How many ordered pairs of whole numbers [m,n][m, n][m,n] satisfy m2+n2−mn≤3m^2 + n^2 - mn \le 3m2+n2−mn≤3? First, narrow down the domain, and then use systematic search to come up with an exhaustive list. (Hint: there should be 131313 pairs in your list)

Solution

To narrow down the domain, we first note that (m+n)2=m2+2mn+n2(m + n)^2 = m^2 + 2mn + n^2(m+n)2=m2+2mn+n2 and (m−n)2=m2−2mn+n2(m - n)^2 = m^2 - 2mn + n^2(m−n)2=m2−2mn+n2. Therefore, after some simplification, we get 14(m+n)2+34(m−n)2=m2+n2−mn \frac{1}{4}(m+n)^2 + \frac{3}{4}(m-n)^2 = m^2 + n^2 - mn 41​(m+n)2+43​(m−n)2=m2+n2−mn and in particular, since all squares are non-negative, if (m+n)2>3×4=12(m+n)^2 > 3 \times 4 = 12(m+n)2>3×4=12 or (m−n)2>3×43=4(m-n)^2 > 3 \times \frac{4}{3} = 4(m−n)2>3×34​=4 then we cannot possibly satisfy the required inequality. Therefore, we know that −3≤m+n≤3-3 \le m+n \le 3−3≤m+n≤3 and −2≤m−n≤2-2 \le m-n \le 2−2≤m−n≤2.

This lets us narrow down the possible values of mmm and nnn to check considerably: we cannot have m≥3m \ge 3m≥3 because then n≥1n \ge 1n≥1 and so m+n≥4m+n \ge 4m+n≥4. Similarly, we cannot have m≤−3m \le-3m≤−3. And because mmm and nnn can be interchanged, we also require the same conditions for nnn. Hence, it suffices to check whole numbers mmm and nnn such that −2≤m≤2-2 \le m \le 2−2≤m≤2 and −2≤n≤2-2 \le n \le 2−2≤n≤2. We can continue to use the conditions from above to narrow down the list further, but there are only 252525 possibilities now.

Now fix some specific values for mmm and consider the possible values for nnn:

  • If m=−2m = -2m=−2, then since m−n≤2m - n \le 2m−n≤2 we must have n=−2n = -2n=−2, n=−1n = -1n=−1 or n=0n = 0n=0. But n=−2n = -2n=−2 is not allowed because then m+n<−3m+n < -3m+n<−3. If n=−1n = -1n=−1, then m2+n2−mn=3m^2 + n^2 - mn = 3m2+n2−mn=3, so this is in fact a solution. If n=0n = 0n=0, then m2+n2−mn=4m^2 + n^2 - mn = 4m2+n2−mn=4 which means this is actually not a solution (remember, although we narrowed the domain, it does not mean that everything in the domain will be a valid solution).

  • If m=−1m = -1m=−1, by a similar argument as above we must have n=−2n = -2n=−2, n=−1n = -1n=−1, n=0n = 0n=0 or n=1n = 1n=1. In the first case, m2+n2−mn=3m^2 + n^2 - mn = 3m2+n2−mn=3 and so this is in fact a solution. In the second case, m2+n2−mn=1m^2 + n^2 - mn = 1m2+n2−mn=1 and so this is also a solution. In the third case, m2+n2−mn=1m^2 + n^2 - mn = 1m2+n2−mn=1 and so this is also a solution. In the fourth case, m2+n2−mn=3m^2 + n^2 - mn = 3m2+n2−mn=3 and so this is also a solution.

  • If m=0m = 0m=0, then n=−2n = -2n=−2, n=−1n = -1n=−1, n=0n = 0n=0, n=1n = 1n=1, or n=2n = 2n=2. Since m=0m = 0m=0, m2+n2−mn=n2m^2 + n^2 - mn = n^2m2+n2−mn=n2 therefore only n=−1n = -1n=−1, n=0n = 0n=0 and n=1n = 1n=1 are in fact solutions.

  • If m=1m = 1m=1, then n=−1n = -1n=−1, n=0n = 0n=0, n=1n = 1n=1 or n=2n = 2n=2. In the first case, m2+n2−mn=3m^2 + n^2 - mn = 3m2+n2−mn=3 and so this is a solution. In the second case, m2+n2−mn=1m^2 + n^2 - mn = 1m2+n2−mn=1 and so this is also a solution. In the third case, m2+n2−mn=1m^2 + n^2 - mn = 1m2+n2−mn=1 and so this is also a solution. In the fourth case, m2+n2−mn=3m^2 + n^2 - mn = 3m2+n2−mn=3 and so this is also a solution.

  • If m=2m = 2m=2, then n=0n = 0n=0 or n=1n = 1n=1. In the first case, m2+n2−mn=4m^2 + n^2 - mn = 4m2+n2−mn=4 and so this is not a solution. In the second case, m2+n2−mn=3m^2 + n^2 - mn = 3m2+n2−mn=3 and so this is in fact a solution.
